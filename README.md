# Fear Less Maths

Worksheet generator for LA Excellence Schools / IDPS Orchards.

## Levels Available (L1-L12)
- Level 1: Number sense & counting
- Level 2: Addition
- Level 3: Subtraction
- Level 4: Multiplication
- Level 5: Division
- Level 6: Fractions
- Level 7: Decimals
- Level 8: Integers
- Level 9: Factors, Multiples, HCF, LCM
- Level 10: Ratio and Proportion
- Level 11: Algebra — Expressions
- Level 12: Algebra — Equations

**Total: 672 worksheets** (14 sublevels × 4 sheets × 12 levels)

Each sheet has exactly 20 questions. A4 format. Print ready.
